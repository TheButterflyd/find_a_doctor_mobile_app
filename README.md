# Find a doctor - Mobile App

Prototype app: A mobile application built with Flutter that helps users find doctors and book appointments.

## Technologies Used:
- Flutter / Dart
- Firebase (Authentication and Database)
- OpenAI API (for intelligent medical assistance)

## Setup and Installation
1. Clone the repository.
2. Create a `.env` file in the project root directory and add your API key:
   `OPENAI_API_KEY=your_api_key_here`
3. Run `flutter pub get` to install dependencies.
4. Run `flutter run` to start the application.
